# remember

## Build

### Requirements

* [Racket 7.5+](https://racket-lang.org/)
* macOS Catalina
* Xcode 11+

### First-time Setup

    $ raco pkg install --name remember core/

### Building

    $ make
    $ $(cd cocoa/remember && xcodebuild)
